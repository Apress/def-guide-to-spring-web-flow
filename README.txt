The Definitive Guide to Spring Web Flow - Source Code
=====================================================


The source code for the complete examples described in the book can be found here:


Chapter09/springbank

	Contains the complete source code for the "Spring Bank" sample application
	referenced in most chapters of the book and in Chapter 9, The Sample Application
	in particular.


Chapter11/swfextensions

	Contains the source code for the Spring Web Flow extensions described in
	Chapter 10, Real-World Use Cases and Chapter 11, Extending Spring Web Flow.


Both the springbank and swfextensions projects use the Spring Jumpstart build system.
Please refer to chapter 2, Getting Started, and Chapter 9, The Sample Application,
for more details on Spring Jumpstart and how to build this code for use in your own
applications.
